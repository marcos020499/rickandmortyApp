# 🚀 Quick Start

💿 Install all dependencies:

```sh
cd rickandmortyApp
npm install


```

🚴‍♂️ Run your App:

IMPORTANT:

- Android:
  - Command 
    `react-native run-android` for react-native-cli package users or 
    `npx react-native run-android` if you're using the latest recommended version of node.js
